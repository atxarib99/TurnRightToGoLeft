# Car Select

Car Select is a replay companion app. When watching a race replay with multiple drivers, it shows a vertical list of every car in the session so you can instantly jump to any driver's perspective with a single click.

## Features

- **One-click camera switch** — click any driver row to focus the replay camera on that car
- **Color-coded model stripes** — each car model gets a unique color stripe on the left, making it easy to spot teammates or cars of the same model at a glance
- **Sort by Slot** — lists cars in their original session order
- **Sort A-Z** — lists cars alphabetically by last name, useful when hunting for a specific driver
- **Active car highlight** — the car you're currently watching is highlighted in orange
